# GLEEBOI — Portfolio (Simple Deploy Version)

This is a **single-file version** of the portfolio — all CSS and JavaScript are built directly into `index.html`, so there are no `css/` or `js/` folders to worry about.

Only two things need to go into your GitHub repo:
1. `index.html` — just drag this one file in.
2. `assets/` folder — contains your CV and certificate PDFs. Drag the whole `assets` folder in as one piece.

See the deployment message in chat for exact step-by-step instructions.
